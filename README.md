# dnsserver-x86
This project redirect the data to it's dns-server and get the domain's ip back.


## Usage
### Install project :
    ./dnssever-control.sh install [port]

### Uninstall project :
    ./dnssever-control.sh uninstall

### Restart project :
    ./dnssever-control.sh restart

### Get status
    ./dnssever-control.sh status
